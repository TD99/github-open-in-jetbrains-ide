GitHub to JetBrains IDE

Version 1.0.0

Quickly open GitHub repositories in your favorite JetBrains IDE.

1. Supported IDEs

All major JetBrains IDEs are supported, including IntelliJ IDEA, PyCharm, WebStorm, PhpStorm, CLion, GoLand, Rider, RubyMine, DataGrip, and DataSpell.

2. Using the extension on GitHub

On a GitHub repository page, open the Code dropdown menu.

You’ll see a new button:

  • "Open with <IDE>": opens the repository directly.
  • "▼"-button: lets you pick another IDE.

3. Default IDE

The IDE you pick from the "▼"-menu becomes your new default. The extension remembers your choice.

4. Toolbox Behavior

If a repository was already cloned in a JetBrains IDE, that IDE will be used automatically (instead of the default set in the extension). This is JetBrains Toolbox’s intended behavior and can be useful to automatically locate the repo clone on your device.
